# 🏠 Property Fish - Manual Testing Project

## 📌 Project Overview

This project involves manual testing of the Property Fish application, which is used for searching and managing real estate properties.

## 🎯 Objectives

* Verify property search functionality
* Validate filters (price, location, property type)
* Test user login and registration
* Ensure smooth navigation across pages

## 📄 Test Artifacts

* Test Plan
* Test Scenarios
* Test Cases

(All documents are maintained in a single Excel file)

## 🛠 Tools Used

* Microsoft Excel
* Word

## 🔍 Testing Type

* Functional Testing
* UI Testing
* Regression Testing
